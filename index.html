EPS Calculation Logic


import pandas as pd

def calculate_eps(net_income, preferred_dividends, shares_outstanding):
  """Calculates Earnings Per Share (EPS)."""
  return (net_income - preferred_dividends) / shares_outstanding

# Example for Q3
eps_q3 = calculate_eps(250_000_000, 5_000_000, 140_000_000)

Forecasting: CAGR Formula

We use the Compound Annual Growth Rate (CAGR) for projections:

\[ \text{CAGR} = \left( \frac{\text{Ending Value}}{\text{Beginning Value}} \right)^{\frac{1}{\text{n}}} - 1 \]

Where 'n' is the number of years.
